## Description
This project is developed for the Data Communication course.
It demonstrates socket programming using Python with a client-server architecture.

## Project Structure
- Client 1: Sender
- Server: Error injector / corruptor
- Client 2: Receiver
- Tkinter-based GUI files included

## Files
- client1_sender.py
- Client1_TK.py
- client2_receiver.py
- Client2_TK.py
- server_corruptor.py
- Server_TK.py
- main_app.py

## How to Run
1.Run 'main_app.py'
or
-if it doesn't work-
1. Run `server_corruptor.py`
2. Run `client2_receiver.py`
3. Run `client1_sender.py`

## How to Use
1.Start Client2.
2.In Server,we choose the method and press 'Start Server'.
3.In Client1 we choose the method and write the text we want,before pressing 'Send'.
4.We check Server.
5.We check Client2 for result.
6.If restart is wanted,then back to step 1.

## Technologies
- Python 3
- Socket Programming
